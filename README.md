# GOLD PICK 웹앱 프로토타입

금거래소 추천, 예상 견적, 방문 예약, 골드바 구매 예약, 커뮤니티와 MY 화면을 포함한 동작형 프론트엔드 시안입니다.

## 실행 방법

### 가장 간단한 방법
`index.html`을 브라우저에서 엽니다. 대부분의 기능은 바로 작동합니다.

### 앱 설치(PWA) 기능까지 확인
폴더에서 간단한 웹서버를 실행합니다.

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080`에 접속하면 됩니다.

## 포함 기능
- 오늘의 금시세 홈
- 순도·중량 기반 예상 매입가 자동 계산
- 제품 사진 미리보기
- 추천 금거래소 필터·검색
- 매장 상세 및 방문 예약
- 골드바 상품 목록·가격 잠금·구매 예약
- 골드톡 게시글 작성
- MY 견적·예약·주문 저장
- 가맹점 관리자 대시보드 예시
- 브라우저 LocalStorage 저장
- PWA 매니페스트 및 오프라인 캐시
